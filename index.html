Seller Tools

Personal shop management utilities

This application provides personal seller tools for managing an shop. It connects to the API to help with listing management, inventory tracking, and order monitoring — used solely by the shop owner for their own store.

No user data is collected or shared. This is a single-user private tool with no public-facing functionality.

Listing manager

Bulk update and manage active listings

Order tracker

Monitor and export order history

Inventory sync

Sync stock levels with print providers

Privacy & data use

This tool accesses only the authenticated seller's own shop data via the API. No data is stored on external servers. All processing is local. This application is not available to the public and is not a commercial product.
